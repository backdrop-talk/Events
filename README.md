# Events
Talk about upcoming virtual and in-person events. Come help us plan or help make it happen!

Events are a big part of Backdrop and we can always use some fresh ideas and volunteers to help make things happen. Jump in the disucssion and get involved!

## Where to start...

1) If you want to discuss things, head over to the **Discussions** tab.

2) If you want to add or ammend a policy, create an issue in the **Issues** tab.

3) If you want to see the progress of any outstanding issues, check out the **Projects** tab.
